# Mini ML

Implemented ridge linear regression and L2-regularized logistic regression from scratch using `python` and `numpy`

## Getting started
1. Install `numpy` (e.g., `pip install numpy`).
2. Import `LinearRegression` or `LogisticRegression`, tune `learning_rate`, `nb_of_iterations`, and `lambda_` as needed, then call `fit(x, y)` and `predict(x)` to explore behavior on your data.
